## Methodology notes

- Window: 90 days where viable, expanded to 180 or 365 for low-cadence profiles (noted per file)
- Original-vs-repost separation: applied per spec where data was visible; LinkedIn's screenshot UI did not always show share-specific repost engagement, limiting Amplification Gap math at the per-share level (noted in Alexi profile and findings)
- Voice classifications (ICP buyer / coach / internal / adjacent / personal) were applied based on visible LinkedIn headlines and titles
- Vision-parsed from PDF screenshots (poppler available); per-page sample size was usually first 5 pages of each profile's posts PDF
