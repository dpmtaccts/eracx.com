# ERA v4 components

Components for the `/v4` route at eracx.com.
Aesthetic: Bloomberg / Turley brutalist — Anton display, Archivo Black blocks,
IBM Plex Sans body, JetBrains Mono labels. Section grounds in white, magenta
(--v4-hot), yellow, cobalt, and ink.
Tokens at `src/styles/v4-tokens.css`, component styles at `src/styles/v4-components.css` —
both scoped under `.v4-root` so they never bleed into other routes.
Do not reuse these components outside `/v4`.

## Render order in V4.tsx

V4Statement → V4Warmth → V4WhatEra → V4Evidence → V4HowItWorks → V4Technology
→ V4Lab → V4FAQ → V4Footer

## Components

- **V4Wordmark.tsx** — Small inline ERA wordmark (paths 2–4 of public/images/era-logo.svg, viewBox `6744 0 13458 4387`, no symbol, fill="currentColor"). Reused by V4Header (small, in every section's issue bar) and V4Statement's publication mast (enlarged, page-top).
- **V4Header.tsx** — Bloomberg-style issue bar at the top of every section. Takes a `phase` string (left side, dot-separated) and a `meta` array (right side). Renders the V4Wordmark at the end of the right group as a magazine-style running head — color inherits from the section so it flips white/ink across grounds. Mono uppercase, 0.14em letter-spaced.
- **V4Nav.tsx** — Main navigation. Inline ERA symbol mark (single path from public/images/era_symbol.svg, fill="currentColor", height 32px) on the left, mono-uppercase links in the middle (Warmth, Evidence, How it works, Tech, Lab), solid currentColor "Get started" CTA on the right that opens a `mailto:hello@eracx.com`. Border-bottom of 1px currentColor.
- **V4Statement.tsx** — §01 hero / statement section. ONE` / `GTM SYSTEMS · EST. 2024 · wordmark`), V4Nav, two-column hero (massive Anton headline "Cold / outbound / is over." left, ink-bordered sidebar with eyebrow, lede, "Evaluate your GTM →" CTA right), and a 4-cell meta band (5 / 9 / 24 / 80%) with top and bottom 1px ink rules.
